1. Send email notification on detecting any change in the directory:

To send email notifications, we can use the `smtplib` module in Python. We need to provide the SMTP server details, sender and receiver email addresses, subject, and message body.

```python
import smtplib
from email.mime.text import MIMEText

def send_email(subject, message, sender_email, receiver_email, smtp_server, smtp_port, username, password):
    msg = MIMEText(message)
    msg['Subject'] = subject
    msg['From'] = sender_email
    msg['To'] = receiver_email

    try:
        with smtplib.SMTP(smtp_server, smtp_port) as server:
            server.starttls()
            server.login(username, password)
            server.sendmail(sender_email, receiver_email, msg.as_string())
    except Exception as e:
        print(f"Failed to send email: {e}")
```

We can call this function whenever a change is detected in the directory. For this, we need to keep track of the previous state of the directory and compare it with the current state. We can use the `os` module in Python to get the list of files in the directory and their modification time.

```python
import os
import time

def get_directory_state(directory_path):
    state = {}
    for root, dirs, files in os.walk(directory_path):
        for file in files:
            path = os.path.join(root, file)
            state[path] = os.stat(path).st_mtime
    return state

def detect_directory_changes(directory_path, previous_state):
    current_state = get_directory_state(directory_path)
    changes = []
    for path, modified_time in current_state.items():
        if path not in previous_state:
            changes.append((path, 'created'))
        elif modified_time != previous_state[path]:
            changes.append((path, 'modified'))
    for path in previous_state:
        if path not in current_state:
            changes.append((path, 'deleted'))
    return current_state, changes

previous_state = get_directory_state('path/to/directory')
while True:
    current_state, changes = detect_directory_changes('path/to/directory', previous_state)
    if changes:
        print(f"Changes detected: {changes}")
        # send email notification here
    previous_state = current_state
    time.sleep(1)
```

This code will keep checking for changes in the directory every second and send an email notification whenever a change is detected. We can call the `send_email` function with appropriate arguments to send the notification.

```python
def detect_directory_changes(directory_path, previous_state, filter_func=None):
    current_state = get_directory_state(directory_path)
    changes = []
    for path, modified_time in current_state.items():
        if path not in previous_state:
            if not filter_func or filter_func(path):
                changes.append((path, 'created'))
        elif modified_time != previous_state[path]:
            if not filter_func or filter_func(path):
                changes.append((path, 'modified'))
    for path in previous_state:
        if path not in current_state:
            if not filter_func or filter_func(path):
                changes.append((path, 'deleted'))
    return current_state, changes
```

4. Integrate the script with machine learning algorithms to detect anomalies in the directory changes:

To detect anomalies in the directory changes, we can use machine learning algorithms such as clustering or anomaly detection. We can use libraries such as scikit-learn or PyOD for this purpose.

```python
from sklearn.cluster import KMeans
import numpy as np

def detect_anomalies(changes):
    files = [path for path, status in changes if status in ['created', 'modified']]
    sizes = [os.path.getsize(file) for file in files]
    X = np.array([[size] for size in sizes])
    kmeans = KMeans(n_clusters=2).fit(X)
    labels = kmeans.labels_
    return [files[i] for i, label in enumerate(labels) if label == 1]
```

This code uses KMeans clustering to separate the files into two clusters based on their size and returns the files in the second cluster, which are considered anomalies.

We can call this function after `create_notification_message` and include the anomalies in the email message body.

